## Base ambient LED control (groups 0x20/0x22 — 32/34)

Controls 2 LED strips (9 LEDs each) on wheelbase body. Group 32 = write, group 34 = read. Sent to main device (`0x12`). Found in `rs21_parameter.db` but not observed in USB captures.

Full command table: [`../devices/main-hub-0x12.md` § Group `0x20` / `0x22`](../devices/main-hub-0x12.md#group-0x20--0x22-32--34--base-ambient-leds).
